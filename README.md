# Instance Manager MVP Frontend

This is a safe simulation frontend MVP for a local Roblox multi instance coordination dashboard.

It does not include Roblox automation, credential collection, exploit code, anti cheat bypasses, or backend control.

## Run locally

```bash
cd frontend
npm install
npm run dev
```

Open this URL:

```text
http://localhost:3000
```

## What works today

The dashboard includes overview cards, account state tracking, selected account checklist, crew invite simulation, queue display, logs, server groups, performance metrics, settings, and responsive layout.

The Join Crew button only changes local simulated UI states.
